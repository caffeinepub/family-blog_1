# Specification

## Summary
**Goal:** Add like and comment functionality to blog posts.

**Planned changes:**
- Add like counter to each blog post with increment function
- Add comments array to each blog post with ability to add new comments
- Display like count and like button on post detail page
- Display all comments in chronological order on post detail page
- Add comment form for submitting new comments
- Show like count preview on home page for each post
- Create React Query hooks for like and comment operations

**User-visible outcome:** Users can like blog posts and leave comments with their name. Like counts and comments are displayed on both the home page and individual post pages.
